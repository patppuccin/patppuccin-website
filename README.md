# Patppuccin Website

This is the source code for the [Patppuccin](https://patppuccin.com) website.

## Commit Conventions

Commits follow the Conventional Commits specification.  

The required structure is: `type(scope): description`

**Allowed Commit Types**

- feat      – introduce new features or enhancements
- fix       – correct bugs or unexpected behavior
- refactor  – restructure code without altering functionality
- chore     – maintain tooling, configs, CI, scripts or build steps
- update    – modify or improve existing content or assets

**Allowed Commit Scopes**

- content   – changes to site content, assets, structure or organization
- core      – core logic, generators, scripts and internal functionality
- config    – VitePress configuration, theme adjustments or UI updates
