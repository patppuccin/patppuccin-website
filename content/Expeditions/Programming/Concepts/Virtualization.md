# Basics of Virtualization

## What is Virtualization

Virtualization is a technology that uses software to create an abstraction over the computer hardware. This allows the resources of a single computer to be shared by many computers, called as [Virtual Machines](./Virtual-Machines.md) or VMs.

Virtualization allows for more efficient utilization of the hardware and allows a greater return on the company/organization's investment. It is the technology that drives [Studio/Expeditions/Cloud Engineering/Cloud Technology](/Studio/Expeditions/Cloud%20Engineering/Cloud%20Technology).

## Hypervisors

[Hypervisors](/Hypervisor) facilitate and coordinate the creation and management of VMs on a computer hardware.
